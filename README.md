# Simple Interest Calculator

A shell script to calculate simple interest.

## Description

This project provides a bash script that calculates simple interest given a
principal amount, annual rate of interest, and time period in years.

**Formula:**

```
Simple Interest = (Principal x Rate x Time) / 100
```

## Getting Started

### Prerequisites

- Bash shell (Linux / macOS) or Git Bash (Windows)

### Usage

```bash
bash simple-interest.sh
```

Follow the prompts to enter:
1. Principal amount
2. Rate of interest (per annum)
3. Time period in years

The script will output the calculated simple interest.
